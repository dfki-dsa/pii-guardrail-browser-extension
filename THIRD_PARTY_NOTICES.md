## Included Legal Files

Every official release package must include these files at package root:

- `LICENSE`

`*.LICENSE.txt` sidecar files for bundled third-party JavaScript must also
remain in `dist/` and in the final release ZIP.

## Rust And WebAssembly Components

Privacy Guardrail includes a deterministic recognizer compiled from Rust to
WebAssembly. The Rust dependency versions are recorded in `crate/Cargo.lock`.
The direct runtime dependencies declared in `crate/Cargo.toml` are:

- `js-sys`
- `regex`
- `serde`
- `serde_json`
- `wasm-bindgen`

The release package contains the compiled WebAssembly output, not vendored Rust
source. Existing crate copyright, license, and attribution notices must be kept
intact when source-form redistribution is added in the future.

## BardsAI Model Assets

Official public beta packages use the BardsAI EU multilingual PII anonymization
model:

- Model ID: `bardsai/eu-pii-anonimization-multilang`
- Provider: bards.ai
- License: Apache-2.0
- DOI: `10.57967/hf/8721`
- Source: https://huggingface.co/bardsai/eu-pii-anonimization-multilang

The package includes prepared runtime files under
`models/ner/bardsai-eu-pii-anonimization-multilang/`:

- `config.json`
- `tokenizer.json`
- `tokenizer_config.json`
- `onnx/model_q4f16.onnx`
- `onnx/model_q4f16.onnx.data`

The `model_q4f16.onnx` and `model_q4f16.onnx.data` files are modified/generated
Privacy Guardrail release artifacts. They are produced from upstream BardsAI
model files by the repository's conversion scripts using ONNX Runtime
MatMulNBits 4-bit weight-only quantization and ONNX external-data packaging.
They are not upstream-original BardsAI files.

The active conversion manifest in the release preparation workspace records:

- Method: `converted-float-to-fp16-then-q4f16`
- Quantization: 4 bits, block size 32, symmetric weights, `QOperator` format,
  `MatMul` operators
- `model_q4f16.onnx` SHA-256:
  `3e74fb3e23b59302d850f1cbbf12949d3bd77e9ae8890d866ed884f377b9c59c`

Inactive or intermediate model artifacts such as `model_fp16.onnx`,
`model_fp16.onnx.data`, `model_quantized.onnx`, conversion manifests, and local
source caches are excluded from official release ZIPs by the packaging rules.

## Non-Packaged Model Assets

The repository contains scripts for preparing or benchmarking additional model
sources, including HikmaAI and AI4Privacy assets. These assets are not part of
the official public beta runtime package unless the release process is changed
explicitly.

AI4Privacy assets are treated as local research/prototype assets in this
repository and must not be included in an official package unless their license
and the intended distribution purpose have been reviewed for that release.
